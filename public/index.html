Cybersecurity Resume Builder

An AI-driven engine for ATS-ready cybersecurity resumes. Run npm start to generate a modern layout sample.

Features

  • Role-aware keyword prioritization for cloud, SOC, red team, GRC, and more.
  • ATS scoring with coverage, impact, readability, and role relevance.
  • Template-based rendering for HTML with pluggable layouts.

Usage

  1. Install dependencies: npm install
  2. Generate sample: npm start
  3. Open output/resume.html in your browser.
